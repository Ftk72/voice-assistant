# Handoffs

Convention propre à ce projet : tous les handoffs de session sont versionnés ici, numérotés dans l'ordre (`0001-…`, `0002-…`).

**Seul le dernier (numéro le plus élevé) fait foi.** Les précédents ne sont conservés que comme historique — ne pas s'y intéresser pour reprendre le travail.

À la fin de chaque session, générer le handoff suivant via le skill `/handoff` et l'enregistrer ici avec le numéro suivant.
